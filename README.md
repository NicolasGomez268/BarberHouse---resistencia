# Peluqueria App

Monorepo para una aplicacion de gestion de peluqueria.

## Stack
- Frontend: React 18 + Vite + TypeScript + Tailwind CSS
- Backend: Node.js + Express + TypeScript
- DB: Firebase Firestore
- Auth: Firebase Authentication
- Validacion: Zod

## Estructura
- `frontend/`: aplicacion web.
- `backend/`: API Express con Clean Architecture por modulo.

## Desarrollo
```bash
cd frontend
npm run dev
```

```bash
cd backend
npm run dev
```
